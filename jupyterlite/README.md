# JupyterLite notes

The full notebooks use geopandas, rasterio, rioxarray, and GDAL-backed libraries.
Those packages are usually not available in a plain JupyterLite/Pyodide browser
runtime.

Use `notebooks_lite/` for browser-friendly teaching or demonstration notebooks.
They implement the same risk-score logic on synthetic tables and can be adapted
for CSV exports generated by the full desktop/server workflow.

Suggested deployment pattern:

1. Run full geospatial notebooks in JupyterLab or GitHub Codespaces.
2. Export lightweight CSV/GeoJSON summaries.
3. Load those summaries into JupyterLite notebooks for interactive review.
